DETO_3D User Manual

Welcome to DETO_3D, a powerful multiscale topology optimization tool employing the discrete element method. This software is designed to facilitate the creation and optimization of structures or systems, emphasizing discrete behaviors.

Getting started

The code makes extensive use of LAMMPS objects and syntax therefore the user is directed towards the LAMMPS offical documentation to for a deeper understanding of some of the essential concepts in the operation of DETO_3D however this Manual is intended to provide all the necessary information for operation of the code.
